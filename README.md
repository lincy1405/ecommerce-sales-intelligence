# E-Commerce Sales Intelligence

An end-to-end business analysis project on 1M+ rows of real UK e-commerce 
transaction data (2009–2011).

## Business Questions
1. Which countries drive the most revenue — and are they also most profitable?
2. What does month-over-month customer retention look like, and when do we lose them?
3. Which products have the highest return rate — should we delist them?
4. What does customer segmentation look like using RFM analysis?
5. When should we run promotions — best time of year and day of week?

## Tools & Stack
- **Python** (Pandas, Matplotlib, Seaborn) — EDA & KPI derivation
- **SQL** (SQLite) — business queries & data exploration
- **Tableau Public / Streamlit** — interactive dashboard
- **Excel** — raw data source

## Project Structure
